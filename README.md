
## Pezesha Microservice with REST API for Money Transfer between Accounts
## Task: 1
### Technologies Used
1. Spring Boot 3.8.8
2. Java 17
3. Postgresql
4. Maven 3

----------------------------------------------------------------------------------
### Testing Instructions
open terminal and git clone the project into your local machine.
```
git clone https://github.com/koryr/money-transfer.git

```
Setup config properties with correct database credentials.
in your terminal cd to root folder of the project.
```
mvn spring-boot:run  #to build the project.

```
If successful run sql script found on resource/script folder to create default user which will be use to create access Token
```
```
Use postman to access API endpoint on localhost:8080

## API

## End-point: Login
Login REST API
### Method: POST
>```
>localhost:8080/auth/login
>```
### Body (**raw**)

```json
{
    "username": "root",
    "password": "admin123"
}
```
